MySQL8.0的新特性

2020-11-07 #mysql

MySQL8.0提供了一些实用的功能特性，这里会一一举例介绍。

通用表达式（Common Table Expressions）

CTE会生成临时表，但只会在查询期间有效，这个临时表在这个查询中可以多次引用和自引用。它的语法：

这个特性会比较适合写一些表中存在递归场景的查询，如：

    # CTE递归生成序列
    WITH RECURSIVE test(n) AS (
        SELECT 1 AS n
        UNION ALL
        SELECT 1+n FROM test WHERE n < 10
    )
    SELECT * FROM test
